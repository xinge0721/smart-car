# 逐飞科技基于CYT2BL3的双电机无刷驱动开源项目

#### Description
逐飞科技精心设计的基于TRAVEO的双电机无刷驱动参考方案，该方案使用CYT2BL3，此单片机具有ARM的M4内核，160Mhz主频。可驱动任意BLDC小型电机、电压为12V。开源项目具有的功能，目前支持电机正反转、支持刹车、支持堵转保护、支持上位机调试。本项目支持FOC及六步换相两种驱动方式，可通过配置文件进行设置。
